Invoice Manager 使い方ガイド

請求書・領収書をアップロードするだけで、AIが自動で内容を読み取り会社と紐付けます

はじめかた — 3ステップ
1
会社マスタを登録する
サイドバーの「会社マスタ」から取引先をあらかじめ登録しておくと、AIが自動で紐付けを行います。CSVで一括インポートも可能です。
2
請求書をアップロードする
「請求書管理」のアップロードエリアにファイルをドロップ、またはボタンから選択します。フォルダごとの一括アップロードにも対応しています。
3
解析結果を確認・修正する
AIが自動で金額・日付・発行者などを抽出します。紐付け会社が違う場合は「詳細」から手動で変更できます。
各機能の説明
ファイルアップロード
  • 対応形式: PDF / JPG / PNG / GIF / WEBP
  • ドラッグ＆ドロップで複数ファイルを一括アップロード
  • 「フォルダを選択」でフォルダ内のファイルをまとめてアップロード（最大50件）
  • 同名ファイルがある場合は上書き確認を表示
  • アップロード中はプログレスパネルで進捗を確認できます
AI 自動解析
  • アップロードすると自動で解析が始まります
  • 請求書番号・発行日・支払期限・金額・発行者などを抽出
  • 会社マスタと照合し、一致度が高い会社を自動で紐付け
  • 解析結果が正しくない場合は「再解析」ボタンで再実行できます
  • 累計トークン数と推定コストを画面上部に表示します
請求書の編集
  • 一覧の「詳細」から解析結果の確認と編集ができます
  • 文書種別（請求書 / 領収書）を手動で変更可能
  • 紐付け会社を変更すると、S3上のファイルも自動で移動します
  • ステータスを「承認済」に変更して管理できます
会社マスタ管理
  • 取引先の会社情報（名前・登録番号・住所・電話・メール）を管理
  • 登録数が多いほどAIの紐付け精度が向上します
  • CSVインポートで一括登録が可能（「サンプルCSV」から書式を確認）
  • テキスト検索で会社名・登録番号を素早く絞り込み
検索・フィルター
  • ファイル名・発行者名でリアルタイム検索
  • ステータス（解析済 / 承認済）で絞り込み
  • 「CSV」ボタンで現在の絞り込み結果をダウンロード
  • 1ページ20件表示、ページネーションで移動
注意事項
  • 削除した請求書は元に戻せません
  • アップロードしたファイルはAI解析のため外部APIに送信されます
  • 1回のフォルダアップロードは最大50ファイルまでです
  • AIの解析結果は必ず内容を確認してご利用ください
ステータスの見かた
解析済 AIによる解析が完了した状態。内容を確認・編集してください。
承認済 内容を確認し、承認済みとしてマークした状態。
